# Google Map Exporter ポータブル版

Node.js同梱のスタンドアロン版です。追加インストールなしで使用できます。

## 起動方法

`起動.bat` をダブルクリックするだけでOKです。

- 自動的にブラウザが開きます（http://localhost:3000）
- 終了するには Ctrl+C を押すか、コマンドプロンプトを閉じてください

## 使い方

### 1. 検索キーワードを入力

検索キーワード欄に **場所 + 検索ワード** を入力します。

**入力例:**
- `東京都 渋谷区 ラーメン`
- `大阪府 梅田 カフェ`
- `新宿駅 居酒屋`
- `横浜市 イタリアン`

### 2. 絞り込み条件（任意）

必要に応じて絞り込み条件を設定できます：

| 項目 | 説明 |
|------|------|
| 評価 | 評価値 + 以上/以下を選択（例: 4.0以上、3.5以下） |
| レビュー数 | レビュー数 + 以上/以下を選択（例: 100件以上） |
| 予算下限/上限 | 価格帯でフィルタリング |
| 住所 | 住所に含まれるキーワード |
| カテゴリ | カテゴリに含まれるキーワード |
| 営業曜日 | 特定の曜日に営業している店舗 |
| 営業時間 | 指定時間に営業している店舗 |
| 取得件数 | 最大取得件数（空欄で無制限） |

### 3. 検索開始

「検索開始」ボタンをクリックすると処理が始まります。

- 進捗状況が表示されます
- 完了すると結果がテーブルに表示されます
- 「CSVダウンロード」ボタンでデータを保存できます

### 4. 結果の確認

結果テーブルには以下の情報が表示されます：

- 店名
- カテゴリ
- 評価・レビュー数
- 予算
- 住所
- 営業時間
- 口コミ
- MAP（Google Mapsへのリンク）

## デバッグモード

`Ctrl+Shift+D` を押すとデバッグモードが有効になります。
「DEV」チェックボックスにチェックを入れると、スクレイピング中のブラウザを表示できます。

## 注意事項

- Google MapsのUI変更により動作しなくなる可能性があります
- 過度な使用はIPブロックの原因になります
- 個人利用のみでお使いください
- 処理中はブラウザを閉じないでください

## トラブルシューティング

**起動しない場合:**
- ウイルス対策ソフトがブロックしていないか確認
- 別のプログラムがポート3000を使用していないか確認

**検索結果が0件の場合:**
- 検索キーワードを変更してみてください
- 絞り込み条件を緩和してみてください

**処理が遅い場合:**
- 検索結果が多い場合は時間がかかります
- 「取得件数」で件数を制限すると高速化できます
- 絞り込み条件で対象を絞ると効率的です
